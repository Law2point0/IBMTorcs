# Race Engineer

A few examples used when making the race engineer. Other examples can be found in previous commits in the `race-engineer-chatbot` branch.

## Overview

**driving_example/** - Example driving code with the race engineer built-in, run `torcs_jm_par.py` in that folder.

**race_engineer_demo.py** - Race engineer using an example snapshot of data from when the TORCS car had crashed.
